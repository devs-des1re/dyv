# Introduction

dyv is a simple and efficient desktop application for downloading videos from YouTube. With its clean and intuitive interface, you can easily download your favorite YouTube content for offline viewing.

**Note:** Make sure [FFmpeg](https://www.ffmpeg.org/download.html) is installed and added to your path, on your system. If you don't know how to install FFmpeg, use this [YouTube Video](https://www.youtube.com/watch?v=orY6lzJbl4w) to help.

# Installation

1. **Go to Releases**: Visit the [Releases page](https://github.com/your-username/dyv/releases) of the dyv repository.
2. **Download Installer**: Download the latest `.exe` installer from the latest release.
3. **Run Installer**: Double-click the downloaded `.exe` file and follow the installation wizard.
4. **Launch dyv**: Once installed, you can find dyv in your Start Menu or Desktop shortcut.

# How to Use

### Step 1: Copy YouTube URL
- Go to YouTube and find the video you want to download.
- Copy the video URL from your browser's address bar.

### Step 2: Paste URL in dyv
1. Launch the dyv application.
2. In the "Enter Youtube URL..." field, paste the copied YouTube URL.

### Step 3: Choose Download Options
- **Location**: Click the "Location" button to select where you want to save the downloaded video.
- **Video**: This section shows the video title and details (not interactive in the UI shown).
- **Download**: Click the "Download" button to start downloading the video.

### Step 4: Monitor Download Progress
- The application will show download progress and notify you when the download is complete.

# System Requirements

- Windows 10 or later
- Internet connection
- Sufficient disk space for downloaded videos

# Troubleshooting

If you encounter any issues:
1. Make sure you have a stable internet connection
2. Verify the YouTube URL is correct and accessible
3. Check that you have write permissions in the selected download location
4. Ensure you have the latest version of dyv installed
